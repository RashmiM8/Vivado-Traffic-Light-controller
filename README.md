# Vivado-Traffic-Light-controller

This project implements a traffic light controller using Verilog HDL and was developed in Xilinx Vivado.

## Files
- **src/** → Verilog source files (design code)
- **tb/** → Testbench files
- **sim/** → Waveform configuration file (.wcfg)

## How to Run
1. Open Xilinx Vivado.
2. Create a new project.
3. Add the source files from `src/`.
4. Add the testbench files from `tb/`.
5. Run simulation and open waveform (`.wcfg` if needed).

## Features
- Implements a basic traffic light controller.
- Includes testbench for verification.
